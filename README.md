# CS+ Social Good Data Science Pipeline Workshop

A reusable CS+ Social Good workshop that introduces the core steps of the data science pipeline using Titanic passenger survival data.

This repository is designed for club workshops, teaching demonstrations, and beginner-friendly introductions to data science using Python and pandas.

---

## Workshop Overview

This workshop explores the question:

**How well was the "Women and Children First" rule upheld on the Titanic?**

Participants walk through a structured data science workflow, from defining a research question to analyzing real-world data and interpreting results.

The focus is not just on coding, but on understanding the decisions behind data analysis.

---

## Learning Goals

By the end of this workshop, participants should be able to:

- Understand how a data science problem is defined
- Load and inspect a dataset using pandas
- Identify relevant variables for analysis
- Perform basic data cleaning
- Define comparison groups
- Calculate survival statistics
- Create simple visualizations
- Interpret results in context

---

## Data Science Pipeline Covered

The workshop follows these steps:

1. Define the Question  
2. Load and Inspect the Data  
3. Clean the Data  
4. Define Groups  
5. Analyze Survival Outcomes  
6. Visualize Results  
7. Draw Conclusions
